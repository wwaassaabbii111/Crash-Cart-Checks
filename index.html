Word Template Manager

Checking template...

Monthly Word forms are filled directly into an official .docx crash cart log — no PDF calibration needed. The app reads the day rows (1–31) in the document's table and writes each entry into the matching row automatically.

If the crash cart log is ever revised, simply upload the new Word file below. As long as it keeps the same day-row table layout (days 1–31 in the first column), it will work immediately — no recalibration required.

The active template is the base for everything: Monthly Review preview, Monthly Forms previews, Word downloads, PDF downloads, and Print all render this exact Word form. Uploading or resetting takes effect immediately — no reload needed.

Firebase

Monthly Forms Access

If Monthly Forms shows an archive access warning, add an authenticated rule for the monthly_forms collection in Firebase Console → Firestore Database → Rules: 

match /monthly_forms/{formId} {
  allow read, write: if request.auth != null;
}

The app can still show live forms without this rule, but the saved archive, running-form persistence, and saved Leader Audit metadata require it.
